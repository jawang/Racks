-----Racks on Racks-----


Description: Given an input and output pair, this application locates
the network card that handles their connection.

Instructions:
1. Enter a valid input (1-1024) and output (1-1024)
2. Press <Enter> or click the Go button

Outputs:
- Rack number
- Chassis number (from the top)
- Card number (from the top)
- Detailed instructions
